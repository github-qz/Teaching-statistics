Interactive T-Interval Generator

This app simulates repeated samples from a normal population and constructs one-sample t confidence intervals for the population mean (using the sample standard deviation). The plot scale stays fixed for a given simulated sample set so you can compare interval widths directly. Changing confidence level updates the intervals without regenerating the sample data. Changing μ, σ, n, or number of samples generates a new simulation.
